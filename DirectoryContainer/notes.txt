Данный файл содержит замечания/предложения разработчиков, появившиеся в процессе работы над проектом.

Файл состоит из двух разделов: 
  "Задачи" - вещи, которые надо сделать, потому что:
      а) решено, что надо, 
      б) очевидно, что надо; 
  "Замечания" - замечания и предложения разработчиков
      (в том числе вещи, которые можно сделать, но без которых можно обойтись).

В каждом разделе могут быть выделены подразделы ( например, "Инсталляция" ), в которых собираются записи, относящиеся к отдельным аспектам разработки. 
При объявлении подраздела рядом c именем подраздела может быть помещен краткий глоссарий.

Файл редактируется любым разработчиком. 

Каждая запись должна быть подписана.

Рекомендуется просматривать данный файл перед каждым сеансом работы.
