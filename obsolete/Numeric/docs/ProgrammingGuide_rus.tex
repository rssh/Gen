\section{ Введение }

 Numeric представляет собой набор компонент для обеспечения расширенной
 функциональности при работе с числовыми данными. На данный момент единственной
 доступной компонентой является класс FixedBigNumber.

 Этот документ представляет собой неформальное описание, полная спецификация
пакета приводится в API reference (\verb|www.gradsoft.kiev.ua/common/ToolBox/Numeric/API/index.html|).

\section{ Требования к программному окружению  }

\begin{enumerate}
  \item переменная препроцессора \verb|HAVE_NAMESPACES| должна быть определена 
  {\em до} включение файла FixedBigNumber.h, если ваш C++ компилятор
  поддерживает пространства имен.
  \item если Вы не пользуетесь MSVC и при этом Ваш компилятор поддерживает тип bool, вам необходимо определить переменную препроцессора \verb|HAVE_BOOL|
\end{enumerate}
