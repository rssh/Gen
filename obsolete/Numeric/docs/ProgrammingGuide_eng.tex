\section{ Programming Environment Conventions }

\begin{enumerate}
  \item if you C++ compiler supports namespaces, preprocesssor variable \verb|HAVE_NAMESPACES| must be defined
  {\em before} inclusion of file FixedBigNumber or FixedBigNumber.h
  \item If you compiler does not have builtin  {\bf bool }, you must define this type,
   with constants {\bf true} and {\bf false } {\em before inclusion } of FixedBigNumber.h 
\end{enumerate}
